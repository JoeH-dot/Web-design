What is CSS?

Cascading Style Sheets (CSS) is a language that can be used to customise the appearance of elements on a web page that uses a markup language. As an example CSS can be used to manipulate html element properties and to make animations, transitions, navbars and drop down menus.

Inline CSS

Inline CSS is one of the ways you can incorporate CSS into your html files. Inline CSS allows you to edit the properties of a single element by typing the CSS code in the html tag of the element you want to edit which allows you to quickly make changes to the style of your website.

Advantages and disadvantages of CSS Implementation

Inline CSS

Advantages of inline CSS is that it allows for CSS to be quickly implemented during the early stages of website development. However the disadvantages of inline CSS is that the code is not reusable and you are only able to edit the properties of one element at a time. You would use inline CSS if you want to quickly test or implement CSS into a few specific elements to see what they would look like. This is because inline CSS can quickly be written within your html files to change the properties of specific elements but can quickly become inefficient if you are implementing CSS into your whole website because you would have to change the style of each element individually.

Embedded CSS

Advantages of embedded CSS is that the code is reusable and it can be used to change the properties of multiple elements at once. However a disadvantage is that all of the code can take up a large amount of space in the html file causing slow loading times. You would use embedded CSS if you are producing a final prototype web page or if you plan on having a specific style for just that html file. This is because whilst you are developing the final prototype you have all of the code in one location and once it has been completed you can easily copy the code into an external css file. Also if you are planning on having a unique style for html files using embedded means you do not have to create multiple external css files which you will have to keep track of.

External CSS

Advantages of External CSS is that the code is reusable and the CSS file can be used to change the properties of elements on multiple html files. However a negative with external CSS is that if the client does not receive the css file then the website will not load properly. You will use external css for completed html files which will all have the same style. This is because external css files can be referenced by multiple html files allowing for them to all have the same style.

CSS Selectors

CSS selectors are used to select specific elements of your html file which you would like to add CSS to. This allows you to create different styles for different elements in your website. There are many different types of CSS selectors, an example of some CSS selectors are class,element and id selectors.

Class selectors will select all elements which have the corresponding class. An example of this would be ".header" which will select all elements in the class named "header".

Element selectors are used to select all of the parts of a website which have a specific element. As an example if you use the element selector "h" this will select all header elements in the html file

Id selectors are used to select elements with a specific id attribute. An example of this would be "#test" which will select all elements with the id attribute "test".

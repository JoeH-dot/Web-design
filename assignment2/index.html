Inline CSS

Inline CSS is one of the ways you can incorporate CSS into your html files. Inline CSS allows you to edit the properties of a single element by typing the CSS code in the html tag of the element you want to edit which allows you to quickly make changes to the style of your website.

Embedded CSS

Embedded CSS is another way you can incorporate CSS into your html files. Embedded CSS makes use of the style element that is put into the HEAD of the document. With embedded CSS you can edit the properties of all elemnts in the html document.

External CSS

External CSS allows you to incorporate CSS into your html files with the use of an external CSS file which is linked to the html file you want to customise.

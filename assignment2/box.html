What is the box model?

Image Source: https://medium.com/@dasjideepak/css-box-model-55f96fb67be9

The Box Model is a design tool used by CSS that is used to customise the layout of html elements. Every html element has a "box" which is split up into different segments. These segments are the Content, Padding, Border and Margin. The Content segment of the box model contains the actual content stored in the element such as videos, images and text. The Padding segment of the box model which is used to create a blank area around the content. The Border is an area around the padding and content segments which allows you to add a border to your elements. The Margin segment wraps around the border and is used to distance different elements from each other.

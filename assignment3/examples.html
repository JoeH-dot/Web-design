Alert, Confirm and Prompt boxes

These popup boxes are very useful in aiding the functionality of the website because they can be used for multiple purposes. As an example the alert box could be used to signify if there has been an error for example if the user has incorrectly inputted data such as a username and password. Alert boxes could also be used to display information such as the results of a calculation to the user. Confirm boxes can be used to make sure that the user wants to perform an action on the website. Prompt boxes could be used as an alternative way to receive user input.

User Redirection

User redirection can be very useful if you are planning on moving to a new domain because normally a user may try to use your old domain name and would be met with an error. But you could have it so that the old domain will redirect the user to the new domain. User redirection can also be useful if you want to redirect the user to sources of information.

Browser Detection

Browser detection can be very important in websites as you can use it to make sure that the users receives a correct version of the website for their device or browser. For example when if the user is on a mobile device you can use browser detection to discover that they are using a mobile browser and then send them the mobile version of the website.
